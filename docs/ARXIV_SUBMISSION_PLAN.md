# arXiv Submission Plan

## Suggested categories

Potential categories to investigate:

```text
math.CO - Combinatorics
math.RT - Representation Theory
math.MG - Metric Geometry
```

The author should choose the category that best matches the final manuscript and any endorser advice.

## Safe title wording

```text
The Green-Schur Bridge for OPAC-018: A Classification-Free Cartan-Schur Proof Candidate of the Root-Polytope Projection Bound
```

## Safe abstract wording

Use `proof candidate`, `referee package`, and `external review pending` until independent review confirms the result.

## Do not say

- solved forever;
- accepted;
- official OPAC solution;
- Millennium Prize result;
- P vs NP/RH/TSP related.

## Before submission

- [ ] Ask at least one algebraic combinatorics reviewer to inspect the roadmap.
- [ ] Confirm citations are complete.
- [ ] Confirm all notation is standard or explicitly defined.
- [ ] Confirm the latest GitHub release is frozen.
