# Cellini-Marietti Transfer Checklist

This checklist exists so referees can verify that the manuscript does not use a vague citation as a black box.

## Checklist

- [ ] State exactly which Cellini-Marietti result is being used.
- [ ] Match notation between the manuscript and the cited result.
- [ ] Verify that the root system hypotheses match: finite, crystallographic, Euclidean.
- [ ] Verify that `U` is a valid Phi-subspace.
- [ ] Explain whether the reduction is codimension-one, arbitrary subspace, or a chain of reductions.
- [ ] Show that the relevant containment/dilation quantity is preserved or bounded through the reduction.
- [ ] Show that no step requires a classification list unless explicitly labeled as audit-only.
- [ ] Separate proof logic from computational audit evidence.

## Recommended manuscript wording

```text
We use the Cellini-Marietti framework only for the stated reduction of the projection-containment problem under the stated hypotheses. The exact local inequality is supplied by the Green-Schur argument, while the finite audit checks arithmetic consistency and is not used as a logical replacement for the reduction theorem.
```
