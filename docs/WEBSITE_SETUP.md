# Website Setup

This repository includes a static website at:

```text
index.html
assets/site.css
```

## Enable GitHub Pages

1. Go to repository Settings.
2. Click Pages.
3. Set source to `Deploy from a branch`.
4. Choose branch `main`.
5. Choose folder `/root`.
6. Save.

GitHub will provide a website URL after the build.

## Website truth boundary

The website intentionally uses `solution-candidate` and `external review pending` wording.
